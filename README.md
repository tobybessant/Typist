# Typist
Typing web-game - race against your friends!

# Build Setup
``` bash
# server
cd server

# install dependencies
npm install

# start with hot reload at localhost:3000
npm run dev

# client
cd client

# install dependencies
npm install

# start with hot reloat at localhost:8080
npm run serve
```
